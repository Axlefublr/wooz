## Building from source

Install dependencies:

* meson (build)
* ninja (build)
* wayland

Then run:

```sh
export CFLAGS="-O3"
meson build
ninja -C build
```

To run directly, use `build/grim`, or if you would like to do a system
installation (in `/usr/local` by default), run `ninja -C build install`.
